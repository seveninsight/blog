每天一个Linux命令（1）：pwd

发表于 2018-12-30 | 更新于 2019-01-03 | 分类于 Linux ， 每天一个Linux命令

pwd是Print Working Directory的缩写。在 Linux 中，用 pwd 命令来查看“当前工作目录”的完整路径。当你在终端进行操作时，如果不太确定当前位置，可以使用 pwd 来查看当前目录在文件系统内的确切位置。

1 功能

pwd是Print Working Directory的缩写。
其功能是显示当前所在工作目录的完整路径。
主要用在当不确定当前所在位置时，通过pwd来查看当前目录的绝对路径。

2 语法格式

【选项说明】
-L：--logical，所在目录是链接时，显示链接目录的路径，(不加参数时默认此方式)
-P：--physical，所在目录是链接时，显示链接目录最终指向的目录的路径。（P是大写字母）

3 示例

常见用法：

例如：

[work@testpc var 16:39:27]$ ll
...
drwxr-xr-x. 13 root root 4096 Dec 30 16:34 log
lrwxrwxrwx.  1 root root   10 Dec  6 21:48 mail -> spool/mail
drwxr-xr-x.  2 root root 4096 Sep 23  2011 nis
...
[work@testpc var 16:40:59]$ cd mail
[work@testpc mail 16:41:03]$ pwd
/var/mail
[work@testpc mail 16:41:04]$ pwd -P
/var/spool/mail
