每天一个Linux命令（3）：ls

发表于 2019-01-02 | 更新于 2019-02-01 | 分类于 Linux ， 每天一个Linux命令

ls命令是list的缩写，用于列出目录中的文件名。
不加参数时，它显示除隐藏文件，以及.和..以外的所有文件及目录的名字，并用不同的颜色区分不同的文件类型。

1 功能

ls 命令用于列出目录中的文件名。
当执行 ls 命令时，默认显示的只有非隐藏文件的文件名。
ls 输出的文件名列表以文件名进行排序，并用不同的颜色表示不同的文件类型。
ls 命令使用频率非常高，参数也非常多，多达四五十个，本文只介绍一部分常用的参数，其他参数可以查看帮助手册。

说明：在linux下，文件名以点.开头表示该文件为隐藏文件，如.bashrc。

2 语法格式

ls [选项] [目录或文件名]

【选项说明】

-a：--all的缩写，显示所有的文件，包括隐藏文件(以.开头的文件)。(常用)
-A：--almost-all的缩写，显示所有的文件，包括隐藏文件，但不包括表示当前目录 . 和上级目录 .. 这两个文件。
-c：时间字段显示 ctime(文件状态的最后改变时间)，默认显示mtime(文件内容的最后修改时间)
-d：--directory的缩写，仅列出目录本身，而不是列出目录里的内容列表。(常用)
-f：直接列出结果，而不进行排序(ls默认会以文件名排序)，也不会用颜色区分文件类型
--color[=WHEN]：是否根据文件类型显示颜色，WHEN可以为never、always或者auto
--full-time：以完整的实际模式显示(包含年月日时分)，类似与ls -l --time-style=full-iso。
-g：列表显示结果，和-l类似，但是不显示文件所属者。
-h：将文件内容大小以GB、KB等易读的方式显示。
-i：结合-l参数，列出每个文件的inode。
-l：列出长数据串，显示出文件的属性与权限等数据信息(常用)，常用别名 ll 代替
-n：和-l类似，只是显示的所属用户和组不是名称而是对应的id。
-r：--reverse，将排序结果以倒序方式显示。
-S：以文件大小排序。
-t：按时间字段排序，默认按照文件名排序
--help：显示帮助信息

3 ls 区分文件类型的颜色

ls会显示出文件的颜色, 系统约定的默认颜色含义如下:

白色：表示普通文件
蓝色：表示目录
绿色：表示可执行文件
红色：表示压缩文件
浅蓝色：链接文件
红色闪烁：表示链接的文件有问题
黄色：表示设备文件
灰色：表示其他文件

颜色的设置，可以在环境变量$LS_COLORS上修改。

4 常见用法与示例

4.1 常见用法

1）列出目录下的文件，不包括.、..、隐藏文件

2）列出文件，包括隐藏文件，但不包括.和..

3）列出所有文件，包括.、..、隐藏文件

4）列出除.、..和隐藏文件 之外的文件，按照文件名排序，并给出权限与属性等相关信息

5）列出文件，按照文件状态的修改时间排序

6）仅列出目录本身，不列出目录里面的文件

7）显示完整时间

8）以人类易理解的方式显示文件大小，如果不加-h选项，默认是按字节显示的。

10）显示文件所属的用户和组的 ID
