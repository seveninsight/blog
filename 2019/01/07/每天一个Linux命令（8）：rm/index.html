每天一个Linux命令（8）：rm

发表于 2019-01-07 | 更新于 2019-01-11 | 分类于 Linux ， 每天一个Linux命令

rm是remove的缩写，rm命令用于删除给定的文件和目录，可以说是 Linux 系统下最危险的命令。
当想要删除当前目录下的所有文件时，如果不小心输入rm -rf . /*，后果自行体会一下。。。

1 功能

rm命令用于删除某个目录下的一个或多个文件或目录。
如果没有使用-r选项，则 rm 不会删除目录。对于链接文件，只是删除这个链接文件，原有文件保持不变。

2 命令格式

rm [选项] [文件或者目录]

【选项说明】

-i：删除文件或目录之前先提示用户
-f：忽略不存在的文件，从不提示
-r或-R：递归删除，将参数中列出的目录全部删除，包括子目录和子文件
-v：显示指令的详细执行过程
--help：显示此帮助信息并退出
--version：输出版本信息并退出

【参数说明】

  • 参数可以是一个或者多个文件或者目录
  • 当参数是目录时，需要-r选项才能删除目录

3 常见用法与示例

3.1 删除文件

rm -f 文件名 文件名 ...

3.2 删除目录

rm -rf 目录名 目录名 ...

3.3 删除以-开头的文件

删除以-开头的文件时，有两种方法：

1、在文件名前加--

rm -- -foo

--选项后面必须是文件名，如果要加其他选项，其他选项必须在--之前。如果不加--选项，会把以-开头的文件误认为是命令选项。

2、在文件名前加上路径

rm ./-foo
